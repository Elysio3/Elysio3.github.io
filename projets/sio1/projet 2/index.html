Forum PHP

  • A4.2.4 – Mise à jour d’une documentation technique

Le projet consiste en la création d'un forum PHP lié à une base de données. Le forum est accessible à partir d'un formulaire de connexion, avec la possibilité de s'inscrire pour ceux qui n'ont pas encore d'accès. La base de données permet l'enregistrement des discussions et commentaires qui sont affichés sur l'espace connecté. L'interface est conçue à l'aide de la librairie Bootstrap pour faciliter sa conception. L'ensemble des requêtes SQL sont assurées en PHP.
